# AI-Based Pothole Detection System 🚧

## 📌 Overview

This project detects potholes using AI and computer vision to improve road safety.

## 🚀 Features

* Upload image/video for pothole detection
* AI-based detection system
* Admin dashboard
* User reporting system

## 🛠 Tech Stack

* Frontend: React
* Backend: Node.js, Express
* AI Model: YOLO / Deep Learning
* Database: MongoDB

## ▶️ How to Run

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend-user
npm install
npm start
